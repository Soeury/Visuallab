# Visuallab - 可视化课程设计

本项目基于 NASA 锂电池老化数据集，对 B0005、B0006、B0007、B0018 四块电池的充放电循环数据进行分析与可视化。项目实现了从原始 .mat 数据解析、关键指标计算，到多维度退化趋势可视化（容量衰减、温升分布、电压曲线演化）以及基础交互仪表板的全流程。

## 目录结构
| 目录/文件 | 说明 |
|--------|------|
| `data/` | 原始 NASA 锂电池 .mat 数据集 |
| `output/` | lab1 清洗后的指标数据（CSV/Excel） |
| `pnt/` | lab2 容量衰减及电压曲线图 |
| `temper/` | lab3 温度分布热力图及相关性分析图 |
| `visual/` | lab4 交互式可视化输出文件 |
| `lab1.py` | 数据解析、清洗与指标计算 |
| `lab2.py` | 容量衰减曲线 + 电压-容量小型多图 |
| `lab3.py` | 温度热力图 + 温升-容量衰减相关性 |
| `lab4.py` | 基础交互设计（悬停数据显示） |
| `README.md` | 项目说明文档 |


## 实验任务说明

### Lab1：数据预处理与关键指标提取
- **输入**：`data/` 目录下的 NASA 数据集（B0005、B0006、B0007、B0018）
- **处理**：
  - 解析 .mat 文件，提取每个电池的充电、放电、阻抗数据
  - 清洗异常值，对齐循环周期
- **输出**：保存至 `output/` 目录
  - 每次循环的**放电容量**
  - **库仑效率**（放电容量 / 充电容量）
  - **平均温度**（放电过程）
  - 其他老化特征参数
- **运行**：`python lab1.py`

### Lab2：容量衰减与电压曲线演化
- **分析内容**：
  1. **容量衰减曲线（折线图）**  
     绘制四个电池随循环次数变化的放电容量曲线，并标注容量降至标称容量 **80%** 和 **70%** 的循环点。
  2. **电压-放电容量小型多图（Small Multiples）**  
     使用子图展示每个电池在不同循环次数下的电压随放电容量变化的曲线，直观呈现曲线形态的衰退过程。
- **输出**：图片保存至 `pnt/` 目录
- **运行**：`python lab2.py`

### Lab3：温度演化与老化相关性
- **分析内容**：
  1. **温度热力图**  
     对每个电池绘制“循环次数 × 放电时间点”的温度热力图，展示放电过程中表面温度随循环次数的变化。
  2. **温升-容量衰减相关性**  
     计算温度上升速率（斜率或峰值温度变化），绘制与容量衰减的散点图并添加趋势线。
- **输出**：图片及统计结果保存至 `temper/` 目录
- **运行**：`python lab3.py`

### Lab4：基础交互可视化
- **功能实现**：
  - 基于 Plotly / Bokeh / Matplotlib 的交互式图表
  - **鼠标悬停显示**：任意循环点的详细数据，包括：
    - 放电容量
    - 表面平均温度
    - 内阻（如可用）
    - 循环次数
- **输出**：交互式 HTML 文件保存至 `visual/` 目录
- **运行**：`python lab4.py`
